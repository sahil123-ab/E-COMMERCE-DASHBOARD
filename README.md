📊 Madhav E-Commerce Dashboard

🔹 Project Overview

This project is an interactive E-Commerce Dashboard created using Power BI.
It helps analyze sales performance, profit trends, customer behavior, and product categories.

---

🔹 Key Features

- 📈 Sales and Profit Analysis
- 📊 Quantity and Revenue Insights
- 🗺️ State-wise Sales Distribution
- 📦 Category-wise Performance
- 💳 Payment Mode Analysis
- 📅 Monthly Profit Trends

---

🔹 Tools & Technologies Used

- Power BI
- Microsoft Excel (Dataset)
- Data Visualization Techniques

---

🔹 Dashboard Insights

- Maharashtra has the highest sales
- Clothing category contributes the most
- COD (Cash on Delivery) is the most used payment mode
- Some months show negative profit trends
